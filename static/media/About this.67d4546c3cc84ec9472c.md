# SpearCoord Portfolio

이 페이지는 **React**와 **Markdown** 파일을 활용하여 만든 **개인 포트폴리오**입니다.

## 주요 기능

- 작성 해둔 문서들을 표시
- 폴더별 구분이 있는 사이드바 네비게이션
- 마크다운 문법 표시

### 적용된 마크다운 문법

1. **굵게** 표시
2. *기울임* 표시
3. ~~취소선~~ 표시

### LaTeX로 수식 추가

슈뢰딩거 방정식 (시간에 대한 독립형):

$$
-\frac{{\hbar^2}}{{2m}} \nabla^2 \psi + V\psi = E\psi
$$

여기서,
$$
\begin{align}
\hbar : 플랑크\,상수\\
m : 입자의\,질량\\
\nabla^2 : 라플라시안\,연산자\\
V : 퍼텐셜\,에너지\\
\psi : 파동\,함수\\
E : 총\,에너지
\end{align}
$$
